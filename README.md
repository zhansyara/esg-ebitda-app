Sustainalyse – ESG Financial Impact Tool
Group

Description
This project contains a Streamlit application called Sustainalyse, which predicts a company’s EBITDA margin based on ESG and financial indicators using a multiple linear regression model.
The model is trained on a cleaned S&P 500 dataset.
Dependencies
Before running the program, ensure the following are installed:
* Python 3.8+
* Required Python libraries:
streamlit
pandas
numpy
scikit-learn
matplotlib
* The dataset file esg_financial_dataset_with_sector.xlsx must be in the same folder as GROUP_06-08_APP.py
Installing
1. Download the project folder containing:
* GROUP_06-08_APP.py
* esg_financial_dataset_with_sector.xlsx
2. Open the folder in your terminal or IDE.
3. Install required packages:
      pip install streamlit pandas numpy scikit-learn matplotlib
Executing Program
1. Navigate to the project folder:
cd path/to/project
2. Run the Streamlit app:
streamlit run GROUP_06-08_APP.py
3. The application will open automatically in your browser.
